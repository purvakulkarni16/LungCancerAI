# LungCancerAI
Project Idea : The main idea of the project is to help a potential cancer patient by
giving him the risk percentage to be diagnosed with lung cancer ,so that he
can change his lifestyle accordingly and avoid cancer. This system can be
used by hospitals to help such potential patients by giving them treatment
and proper advice before they are diagnosed with cancer.

Goals and objective : 
The goal of the project is to build a system that would provide the
probability of a person to be diagnosed with the lung cancer in the
future.
Other objective of the project is to provide patient with the better and
refined view of the cancer causing factors ,so that he/she can change
his/her lifestyle or habits to avoid cancer.
